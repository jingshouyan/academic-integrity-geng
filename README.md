# 耿同学·学术论文造假分析 Skill

> 「孩子，爹今天要教你什么叫学术打假。」

**academic-integrity-geng** 是一个面向 [Reasonix](https://reasonix.ai) 的 **subagent skill**，以「耿同学」——学术圈最严厉的父亲——的人设，从 5 大维度系统扫描学术论文，识别数据造假、图片操纵、方法学缺陷、引用不当及期刊信誉等问题。

所有分析在隔离子 agent 中完成，不占用父会话上下文，使用调用方的默认模型。

---

## 快速安装

```bash
# 方案一（推荐）：从 GitHub 直接安装
reasonix skill install jingshouyan/academic-integrity-geng

# 方案二：手动安装
git clone git@github.com:jingshouyan/academic-integrity-geng.git
cp academic-integrity-geng/Skill.md .reasonix/skills/
```

## 使用方法

```bash
# 分析一篇论文
/run academic-integrity-geng 分析这篇论文: https://example.com/paper.pdf

# 或通过 Tool Call
run_skill({
  name: "academic-integrity-geng",
  arguments: "请分析以下论文全文：\n\n{论文标题}\n{论文内容/链接/DOI}"
})
```

`arguments` 支持传入：
- 论文全文文本
- 公开论文 URL（subagent 会使用 `web_fetch` 获取）
- DOI 号（subagent 会使用 `web_search` + `web_fetch` 定位）

## 输出示例

```markdown
# 🔍 耿同学学术打假报告

> **论文**：Potential therapeutic effects of X in Y disease
> **总体风险评级**：🔴 高

## 一、数据与结果 🔬
- **问题**：p 值全部在 0.01-0.045 之间，无一个 > 0.05
  **证据**：全篇报告 24 个独立比较，p 值范围 0.011-0.043
  **严重度**：🚨 p-hacking 特征显著

## 二、图片与图表 🎨
- **问题**：Fig 2A 和 Fig 3C 的 beta-actin 内参条带完全一致
  **证据**：目测可见条带形状、背景纹理、位置完全重合
  **严重度**：🚨 图片复用（疑似复制粘贴）

...

## 📋 综合评估
**耿总评语**：
> 孩子，爹今天看了篇好论文——p值排得比国庆阅兵还整齐，Western blot剪得比大爷的旧衬衫还拼接。
> 爹不说这是造假，爹只能说这运气好到该去买彩票。做学问先做人，论文可以不惊艳，但不能不要脸。
```

## 风险评级定义

| 评级 | 含义 | 建议行动 |
|------|------|----------|
| 🟢 **低** | 未发现明显学术不端迹象 | 常规引用，无需特别处理 |
| 🟡 **中** | 存在若干可疑点，可能是不严谨而非造假 | 建议联系作者索取原始数据/图片确认 |
| 🔴 **高** | 多个维度发现较严重疑点，数据造假可能性高 | 建议向期刊编辑部发送 Concern |
| ⚫️ **极高** | 系统性造假特征，多个证据链交叉印证 | 建议向期刊+机构正式举报，PubPeer 公开质疑 |

## Caution & Limitations

1. **仅基于文本时无法做图片检测** — 如果用户只提供文本摘要，图像维度只能注明「无法审查」。
2. **不替代专业检测工具** — 耿同学使用的是「基于规则 + 经验的系统审查」，不是 Imagetwin / Proofig 等专门图片检测软件。可疑结果建议用专业工具交叉验证。
3. **误报风险** — 严格的审查标准可能导致假阳性。所有疑点标注了严重度，读者应自行判断。
4. **不构成正式指控** — 所有分析结果仅供学术讨论参考，不应用于法律/人事等正式场合。
